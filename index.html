USER STORIES

MVP GOAL ICEBOX
  • User should be able to log in with Spotify Oauth
  • User can search other spotify users by username and full name
  • User should be able to generate their own communal station based on songs from "circle" Spotify playlists
  • User should be able to play songs from generated station
  • User should be able to get access to their own station once logged in ("circle of one")
  • User should be able to delete circles they have created
  • User should be able to remove other users from circles they have created
  • User should be able to see all circles they are part of
  • User should be able to name their circle they created
  • App should display which user and which users playlist song was pulled from
  • User should be able to skip songs on playlist
  • User should be able to see their own Spotify profile picture
  • While tuned into a station, User should be able to see who else in said station's circle is tuned in
  • PT. 1: User should be able to click "Save Playlist" which will then post a playlist to said users account containing all the songs played so far in that circle
  • PT. 2: Every time a new song is pulled for the circle's station, that song is pushed to the saved playlist in their spotify account
  • User should be able to choose which user playslists should be pulled to play from their circle
  • User should be able to isolate (by genre) songs included in a station's playback
  • User should be able to log in without a spotify account
  • User should be able to select certain user playlists from circle
  • User should be able to leave "circles" they are part of
